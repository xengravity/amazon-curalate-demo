Amazon Pay X Curalate Demo

Interstitial Page

This page will be hosted on the same server as the webstore. This will not be hosted on Curalate or Amazon Pay servers.

  • The user will not see this page
  • The purpose of this page is to take the SKU ID and other parameters from Curalate that were passed through the OAuth2 State Parameters, decode the parameters, and pass them in the URL redirect to the website URL
  • You will need to set the "amazon_Login_accessToken" and "amazon_Login_state_cache" cookies on this page to keep the user session alive on your webstore
  • All code to perform these functions are self contained in the source code of this page

The values that were captured from the previous page are:

SKU ID
Misc Parameter
Environment
Checkout URL
Amazon Seller ID
Amazon Client ID
Access Token
